# Synent Internship Task 1 - Titanic Data Cleaning
 
 Problem Statement
The Titanic dataset contains missing values and duplicate records. The goal of this project is to clean and preprocess the dataset for further analysis.

 Dataset Used
Titanic Dataset

 Objective
- Handle missing values  
- Remove duplicates  
- Rename columns  
- Prepare clean data for analysis  

 Tools Used
- Python  
- Pandas  
- Visual Studio Code  

 Steps Performed
1. Loaded dataset  
2. Checked missing values  
3. Filled null values using mean and mode  
4. Removed duplicate rows  
5. Saved cleaned dataset  

 Results
The dataset was successfully cleaned and prepared for analysis.

 Files Included
- titanic.py  
- titanic.csv  
- cleaned_titanic.csv
